# ☕ Meus Estudos em Java

Bem-vindo! Este repositório é dedicado a documentar o meu progresso e organizar os exercícios práticos do curso de Java. O objetivo aqui é fixar conceitos, testar novas abordagens e construir uma base sólida na linguagem.

## 📚 Tópicos Abordados até o momento
Aqui você encontrará códigos práticos envolvendo:
- Fundamentos da linguagem Java
- Programação Orientada a Objetos (Classes, Atributos, Métodos)
- Expressões Lambda e Programação Funcional
- Composição e encadeamento de funções (Interfaces `Function`, `Predicate`, `BinaryOperator`)
- Boas práticas com tipos de dados (ex: uso de `BigDecimal` para valores financeiros)

## 🛠️ Ferramentas e Tecnologias
- **Linguagem:** Java
- **IDE:** IntelliJ IDEA
- **Versionamento:** Git e GitHub

## 📂 Como navegar
Os arquivos estão organizados por pacotes de acordo com os módulos do curso. Sinta-se à vontade para explorar os códigos!
